1. Download & install xampp
2. Place the entire extract foler into C:\xampp\htdocs
3. Open CMD
4. Type cd C:\xampp\mysql\bin
5. Execute mysql -u root
6. Run create database spm_project
7. Run use spm_project
8. Copy the content in the test.sql to the command line and execute
9. Run the xampp control panel (Apache and MySQL)
10. Type http://localhost/spm_project/index.php in your browser
11. You can run test in the web and test the data in the database
